
This is a simple Scientific Calculator made using Python.  
I created this project to practice Python basics and improve my understanding of functions, conditions, loops, and mathematical operations.

The calculator can perform both normal calculations and some scientific operations using Python’s built-in `math` module.

---

## ✨ Features

### Basic Calculations
- Addition
- Subtraction
- Multiplication
- Division

### Scientific Functions
- Sin(x)
- Cos(x)
- Tan(x)
- Square Root
- Power Calculation
- Pi (π) Value

---

## 🛠️ Technologies Used

- Python 3
- Python `math` module
